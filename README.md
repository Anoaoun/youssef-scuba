# Youssef Scuba Diving Club 🤿

Professional website for Youssef Scuba Diving Club in Damour, Lebanon.

## Features

- 🌊 Bilingual (English & Arabic)
- 📸 Photo Gallery
- 📅 Courses & Trips
- 👥 Team Introduction
- 📞 Contact & Booking
- 📱 Mobile Responsive
- ⚡ Fast & SEO Optimized

## Tech Stack

- **Next.js 14** - React Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **React Icons** - Icons

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/Anoaoun/youssef-scuba.git
cd youssef-scuba
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm run start
```
